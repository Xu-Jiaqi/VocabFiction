# VocabFiction UI 风格规范

## 意象

午后翻开一本纸书的质感。安静、温润、不声张。

## 色彩

| 角色 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#FAF8F3` | 暖米白，模拟纸页 |
| 左气泡 | `#F0EDE5` | 浅杏，他人消息 |
| 右气泡 | `#E8ECF1` | 淡灰蓝，主角消息 |
| 叙述文字 | `#8C8C82` | 温灰，不喧宾夺主 |
| 正文颜色 | `#2C2416` | 墨棕，不是纯黑 |
| 次要文字 | `#A09C94` | 浅温灰，状态/时间/标签 |
| 面板背景 | `#F5F2EB` | 略深米白，与主背景区分 |
| 分割/边界 | `#E5E0D5` | 极淡暖灰 |

- 所有"黑"都是带暖调的深棕，所有"白"都是带暖调的米白
- 不用纯黑 `#000`、纯白 `#FFF`
- 不用冷色调（蓝紫、荧光绿、科技蓝）

## 字体

| 用途 | 字体 | 大小 | 行高 | 字重 |
|------|------|------|------|------|
| 对话正文 | Charter / Georgia | 15px | 1.8 | Regular |
| 叙述文字 | Charter / Georgia | 13px | 1.8 | Regular |
| 角色名 | system-ui sans-serif | 11px | 1.4 | Medium |
| 状态栏标题 | system-ui sans-serif | 14px | 1.4 | Medium |
| 面板标题 | system-ui sans-serif | 20px | 1.5 | Regular |
| 中文正文 | 苹方 / 思源宋体 | 15px | 1.8 | Regular |

- 对话和叙述用衬线体（纸书感）
- 界面元素用系统无衬线体（现代感、可读性）
- 不使用斜体（中文斜体效果差）

## 词汇标记

- **首次出现**：正文加粗 + 紧跟灰色括号释义
  - 示例：**invisible**（隐形的）
  - 释义颜色 `#A09C94`，字重 Regular
- **复习出现**：正文加粗
  - 示例：**footstep**
  - 可点击，点击弹出释义
- **已学会**：普通字重，无标记

不加颜色，不加下划线，不加边框。加粗是唯一的视觉差异。

### 点击查词

复习出现的词（仅加粗，无释义）可点击查词，两级展开：

**第一级：小卡片**

```
                     ╭──────────────╮
  ...the menu was    │  episode     │
  a medical ┌─────┐ │  事件,插曲    │
            │episode│╰──────────────╯
            └─────┘
```

- 点击加粗词 → 词上方浮出小卡片
- 内容：左侧词（加粗），右侧中文释义
- 背景 `#F5F2EB`
- 圆角 8px，极淡阴影
- 动画：原地淡入 0.15s
- 卡片不遮挡当前句子

**第二级：词典面板**

```
┌─────────────────────────────────┐
│                                 │
│  episode                        │  ← 词，加粗，字号较大
│  /ˈepɪsəʊd/                     │  ← 音标
│                                 │
│  1. 事件，插曲                   │  ← 释义（可多义项）
│  2. （人生的）一段经历           │
│                                 │
│  episodes                       │  ← 词形变化
│                                 │
│  "The result sounds like a      │  ← 本集例句
│   medical episode."             │
│                                 │
│  "Each episode builds on the    │  ← 前集例句（如有）
│   last."                        │
│                                 │
└─────────────────────────────────┘
```

- 点击小卡片 → 顶部展开词典面板
- 内容：词、音标、多义项、词形变化、例句（本集 + 历史出现）
- 背景 `#F5F2EB`
- 从顶部向下滑入 0.25s ease-out
- 面板内容超出高度时可上下滑动
- 下方主内容区整体下移，不遮挡当前阅读位置
- 关闭：点击面板外区域，或下拉关闭

## 气泡

### 左气泡（他人对话）

```
┌──────────────────────────────┐
│ Anna                         │  ← 角色名，上方小字
│ ┌──────────────────────┐     │
│ │ Hey.                 │     │  ← 气泡：背景 #F0EDE5
│ │ You're Nukumizu?     │     │     文字 #2C2416
│ └──────────────────────┘     │     内边距 10px/14px
│ 头像                          │     圆角 4px 16px 16px 16px
└──────────────────────────────┘
```

- 头像：圆形 36px，左侧，使用默认占位圆（角色名首字）
- 气泡圆角不对称：左上 4px，其余 16px
- 无阴影

### 右气泡（主角对话）

```
                      ┌──────────────────────┐
                      │              Kazuhiko │  ← 角色名，右对齐
                      │ ┌──────────────────┐ │
                      │ │ ...Nukumizu who? │ │
                      │ └──────────────────┘ │
                      │                  头像 │
                      └──────────────────────┘
```

- 头像：圆形 36px，右侧，使用默认占位圆（角色名首字）
- 角色名右对齐，气泡上方小字
- 气泡：背景 `#E8ECF1`，右对齐，最大宽度 75%
- 圆角：右上 4px，其余 16px
- 无阴影

### 叙述（居中）

```
         I bike to a family restaurant.
         Self-service drinks. Extra-large fries.
```

- 居中，最大宽度 85%
- 颜色 `#8C8C82`，字号 13px
- 上下留白充足（至少 8px 间距）

## 间距

| 元素 | 间距 |
|------|------|
| 消息之间 | 8px |
| 角色切换时 | 16px（左气泡上下换人时额外间距） |
| 叙述与消息之间 | 12px |
| 屏幕左右内边距 | 16px |
| 屏幕上下内边距 | 12px / 20px |

## 面板（集末词汇）

```
┌─────────────────────────────────┐
│                                 │
│       本集读完                  │  ← 温灰小字
│       遇见 3 个词               │  ← 正文，不计数不考核
│                                 │
│   invisible    隐形的            │  ← 新词：正文加粗 + 释义
│   awkward      尴尬的            │
│   footstep                      │  ← 复习：仅加粗
│                                 │
│          继续阅读               │  ← 低调按钮
│                                 │
└─────────────────────────────────┘
```

- 半屏弹出，从底部升起
- 背景 `#F5F2EB`，与主界面有微弱色差
- 措辞用"遇见"而非"学习"
- "继续阅读"按钮为文字 + 箭头，不用实心按钮

## 动画

| 场景 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 新消息出现 | 从下向上淡入 | 0.3s | ease-out |
| 面板弹出 | 从底部滑入 | 0.3s | ease-out |
| 释义浮层 | 原地淡入 | 0.15s | ease-out |

- 不用弹性动效、不用弹跳、不用缩放
- 动画只有两种：slideUp（消息）、fadeIn（面板/浮层）

## 状态栏

```
┌─────────────────────────────────┐
│  The Glass              Ep.3    │  ← 左标题 右集数
├─────────────────────────────────┤
│  ████████░░░░░░░░░░░░░░░░░     │  ← 极细进度条，无数字
└─────────────────────────────────┘
```

- 进度条高度 2px，颜色 `#D4C9B8`
- 不显示百分比数字
- 标题字体 14px，集数 12px
- 背景与主界面同色

## 适配

- 手机全屏；桌面端居中 400px 宽，带圆角边框模拟手机
- 深色模式：暂不提供（方向 A 为固定浅色模式）
