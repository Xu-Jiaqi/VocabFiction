# VocabFiction UI 风格规范

## 意象

午后翻开一本纸书的质感。安静、温润、不声张。

## 色彩

| 角色 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#FAF8F3` | 暖米白，模拟纸页 |
| 左气泡 | `#F0EDE5` | 浅杏，他人消息 |
| 右气泡 | `#E8ECF1` | 淡灰蓝，主角消息 |
| 叙述文字 | `#8C8C82` | 温灰，不喧宾夺主 |
| 正文颜色 | `#2C2416` | 墨棕，不是纯黑 |
| 次要文字 | `#A09C94` | 浅温灰，状态/时间/标签 |
| 面板背景 | `#F5F2EB` | 略深米白，与主背景区分 |
| 分割/边界 | `#E5E0D5` | 极淡暖灰 |

- 所有"黑"都是带暖调的深棕，所有"白"都是带暖调的米白
- 不用纯黑 `#000`、纯白 `#FFF`
- 不用冷色调（蓝紫、荧光绿、科技蓝）

## 字体

| 用途 | 字体 | 大小 | 行高 | 字重 |
|------|------|------|------|------|
| 对话正文 | Charter / Georgia | 15px | 1.8 | Regular |
| 叙述文字 | Charter / Georgia | 13px | 1.8 | Regular |
| 角色名 | system-ui sans-serif | 11px | 1.4 | Medium |
| 状态栏标题 | system-ui sans-serif | 14px | 1.4 | Medium |
| 面板标题 | system-ui sans-serif | 20px | 1.5 | Regular |
| 中文正文 | 苹方 / 思源宋体 | 15px | 1.8 | Regular |

- 对话和叙述用衬线体（纸书感）
- 界面元素用系统无衬线体（现代感、可读性）
- 不使用斜体（中文斜体效果差）

## 词汇标记

- **首次出现**：正文加粗 + 紧跟灰色括号释义
  - 示例：**invisible**（隐形的）
  - 释义颜色 `#A09C94`，字重 Regular
- **复习出现**：正文加粗
  - 示例：**footstep**
  - 可点击，点击弹出释义
- **已学会**：普通字重，无标记

不加颜色，不加下划线，不加边框。加粗是唯一的视觉差异。

### 点击查词

复习出现的词（仅加粗，无释义）可点击查词，两级展开：

**第一级：小卡片**

```
                     ╭──────────────╮
  ...the menu was    │  episode     │
  a medical ┌─────┐ │  事件,插曲    │
            │episode│╰──────────────╯
            └─────┘
```

- 点击加粗词 → 词上方浮出小卡片
- 内容：左侧词（加粗），右侧中文释义
- 背景 `#F5F2EB`
- 圆角 8px，极淡阴影
- 动画：原地淡入 0.15s
- 卡片不遮挡当前句子

**第二级：词典面板**

```
┌─────────────────────────────────┐
│                                 │
│  episode                        │  ← 词，加粗，字号较大
│  /ˈepɪsəʊd/                     │  ← 音标
│                                 │
│  1. 事件，插曲                   │  ← 释义（可多义项）
│  2. （人生的）一段经历           │
│                                 │
│  episodes                       │  ← 词形变化
│                                 │
│  "The result sounds like a      │  ← 本集例句
│   medical episode."             │
│                                 │
│  "Each episode builds on the    │  ← 前集例句（如有）
│   last."                        │
│                                 │
└─────────────────────────────────┘
```

- 点击小卡片 → 顶部展开词典面板
- 内容：词、音标、多义项、词形变化、例句（本集 + 历史出现）
- 背景 `#F5F2EB`
- 从顶部向下滑入 0.25s ease-out
- 面板内容超出高度时可上下滑动
- 下方主内容区整体下移，不遮挡当前阅读位置
- 关闭：点击面板外区域，或下拉关闭

## 气泡

### 左气泡（他人对话）

```
┌──────────────────────────────┐
│ Anna                         │  ← 角色名，上方小字
│ ┌──────────────────────┐     │
│ │ Hey.                 │     │  ← 气泡：背景 #F0EDE5
│ │ You're Nukumizu?     │     │     文字 #2C2416
│ └──────────────────────┘     │     内边距 10px/14px
│ 头像                          │     圆角 4px 16px 16px 16px
└──────────────────────────────┘
```

- 头像：圆形 36px，左侧，使用默认占位圆（角色名首字）
- 气泡圆角不对称：左上 4px，其余 16px
- 无阴影

### 右气泡（主角对话）

```
                      ┌──────────────────────┐
                      │              Kazuhiko │  ← 角色名，右对齐
                      │ ┌──────────────────┐ │
                      │ │ ...Nukumizu who? │ │
                      │ └──────────────────┘ │
                      │                  头像 │
                      └──────────────────────┘
```

- 头像：圆形 36px，右侧，使用默认占位圆（角色名首字）
- 角色名右对齐，气泡上方小字
- 气泡：背景 `#E8ECF1`，右对齐，最大宽度 75%
- 圆角：右上 4px，其余 16px
- 无阴影

### 叙述（居中）

```
         I bike to a family restaurant.
         Self-service drinks. Extra-large fries.
```

- 居中，最大宽度 85%
- 颜色 `#8C8C82`，字号 13px
- 上下留白充足（至少 8px 间距）

## 间距

| 元素 | 间距 |
|------|------|
| 消息之间 | 8px |
| 角色切换时 | 16px（左气泡上下换人时额外间距） |
| 叙述与消息之间 | 12px |
| 屏幕左右内边距 | 16px |
| 屏幕上下内边距 | 12px / 20px |

## 面板（集末词汇）

```
┌──────────────────────────────────────────┐
│                                          │
│              本集读完                     │  ← 温灰小字
│              遇见 5 个词                  │  ← 正文，不计数不考核
│                                          │
│  新词                 旧词                │
│  ────                 ────                │
│  invisible  隐形的    awkward             │  ← 新词：单词 + 释义
│  casual     休闲的    footstep            │      旧词：仅加粗单词
│                      awkward              │
│                                          │
│  (空栏显示虚线占位：                       │
│  ┌ ─ ─ ─ ─ ─ ─ ─ ┐                      │
│    本集无新词          )                  │
│  └ ─ ─ ─ ─ ─ ─ ─ ┘                      │
│                                          │
└──────────────────────────────────────────┘
```

- 集末自动弹出（到达最后一条消息时），非半屏模态
- 目标词按**新词/旧词**左右分栏，`flexDirection: 'row'`
- 新词栏：单词 + 中文释义（`vocabRowNew`），单行截断
- 旧词栏：仅加粗单词（`vocabRowReview`），无释义
- 某一栏无词时显示虚线边框占位符，文字"本集无新词"/"本集无旧词"
- 所有词条行高统一 44px，`numberOfLines={1}` + `overflow: hidden`
- 背景 `#F5F2EB`，与主界面有微弱色差
- 措辞用"遇见"而非"学习"
- 面板底部预留 80px 空间让位给 bottom bar

## 动画

| 场景 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 新消息出现 | 从下向上淡入 | 0.3s | ease-out |
| 面板弹出 | 从底部滑入 | 0.3s | ease-out |
| 释义浮层 | 原地淡入 | 0.15s | ease-out |

- 不用弹性动效、不用弹跳、不用缩放
- 动画只有两种：slideUp（消息）、fadeIn（面板/浮层）

## 状态栏

```
┌─────────────────────────────────┐
│  The Glass              Ep.3    │  ← 左标题 右集数
├─────────────────────────────────┤
│  ████████░░░░░░░░░░░░░░░░░     │  ← 极细进度条，无数字
└─────────────────────────────────┘
```

- 进度条高度 2px，颜色 `#D4C9B8`
- 不显示百分比数字
- 标题字体 14px，集数 12px
- 背景与主界面同色

## 适配

- 手机全屏；桌面端居中 400px 宽，带圆角边框模拟手机
- 深色模式：暂不提供（方向 A 为固定浅色模式）

## 触摸目标

iOS HIG 与 Material Design 均要求交互元素 ≥ 44×44pt 实际可点击区域。本规范在视觉尺寸上保留纸书感的克制，但触摸热区必须达到标准。

| 元素 | 视觉尺寸 | 触摸热区 | 实现 |
|------|---------|---------|------|
| 词汇词 | 13-15px 文字（约 20pt） | ≥ 44pt | 嵌套 Text with onPress（line-height 已扩展到 1.8，热区约 30pt）；后续可改为独立 Pressable 流式布局 |
| 角色头像 | 36px 圆形 | ≥ 44pt | 4px 内边距 = 44pt 容器 |
| 设置图标（导航栏） | 22pt 图标 | ≥ 48pt | 12px H + 12px V padding = 48pt 容器 |
| 段控（设置） | 14px 文字 | ≥ 44pt | `paddingVertical: 12, minHeight: 44` |
| 底部栏箭头 | 28pt 符号 | ≥ 48pt | `paddingVertical: 12, minHeight: 48` |
| 集末词汇项 | 15px 文字 | ≥ 44pt | `paddingVertical: 10, height: 44, numberOfLines: 1` |
| 侧边栏关闭 ✕ | 18pt 文字 | ≥ 44pt | hitSlop={12} + padding 8 |
| 侧边栏集数项 | 14px 文字 | ≥ 56pt | `paddingVertical: 16, minHeight: 56` |

触摸目标之间间距 ≥ 8dp（同向并排时）。

## 动效降级

必须尊重 `prefers-reduced-motion`（iOS Reduce Motion / Android 移除动画）。开启后：

| 场景 | 正常动效 | 降级动效 |
|------|---------|---------|
| 屏幕进入 | 250ms slide-in | 0ms（即时） |
| 消息淡入 | 300ms | 80ms |
| 顶/底栏显示/隐藏 | 200ms | 80ms |
| 词典面板 | 250ms | 80ms |
| 侧边栏 | 220ms | 80ms |

实现：监听 `AccessibilityInfo.isReduceMotionEnabled()` 与 `reduceMotionChanged`，按状态切换时长。

## 动态字号

用户设置的"小/中/大"与系统字号缩放相乘。基础大小仍为 15px / 13px / 11px。

```
effectiveFontSize = baseFontSize × userFontScale × systemFontScale
```

- 系统字号范围 0.85× ~ 1.5×
- 在 1.5× × 用户大（1.2） = 1.8× 时验证布局：消息气泡应能容纳 27px 文字，段间距仍可读

## 系统手势

iOS 边缘右滑返回 / Android 14+ 预测式返回必须与 App 内手势协调：

| 场景 | 行为 |
|------|------|
| 词典面板打开 + 系统返回 | 关闭面板（不退出阅读器） |
| 侧边栏打开 + 系统返回 | 关闭侧边栏（不退出阅读器） |
| 上述都关闭 + 系统返回 | 退出到书架 |
| 移动端顶/底栏隐藏 + 上滑 | 显示栏（已存在） |
| 移动端顶/底栏显示 + 下滑 | 隐藏栏（已存在） |

## 生成加载

书架上处于"生成中"状态的作品显示：
- 集级进度：`生成中 3 / 12 集…`（温灰小字 12px）
- 点击进入生成详情面板（可看到已完成的集和失败的集）
- 后台完成时通过本地通知（`expo-notifications`）提示
- 失败时显示具体错误（网络/认证/超时）并提供"重试"按钮

## 语义色板

除视觉色（背景/气泡/文字/分割）外，引入语义色供状态/反馈使用：

| 令牌 | 色值 | 用途 |
|------|------|------|
| `shadow` | `rgba(44, 36, 22, 0.08)` | 所有阴影（暖棕投影，与暖纸感和谐） |
| `scrim` | `rgba(44, 36, 22, 0.18)` | 模态遮罩（仅用于侧边栏和词典面板的可点击暗层） |
| `success` | `#5B7C5A` | 成功反馈（API 连接成功） |
| `destructive` | `#A04A3A` | 错误反馈（API 连接失败、删除确认） |
| `pressedOverlay` | `rgba(44, 36, 22, 0.06)` | 按下态（比 leftBubble 深一档） |

不使用纯黑 `#000`、纯白 `#FFF`、纯红 `#FF0000`。
